# 📚 BookStore App

A full-stack **BookStore application** built using **React** and **Node.js**.  
This project helps users browse, add, update, and manage books through a clean and responsive interface.

---

## 🚀 Features

- 📖 View list of books
- ➕ Add new books
- 📝 Update book details
- ❌ Delete books
- 🔍 Search and filter books *(can be added if not implemented)*
- 🌐 REST API integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS / Bootstrap / Tailwind

### Backend
- Node.js
- Express.js

### Database
- MongoDB / JSON Server

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/ankitaMahaseth/bookStore.git  
cd bookStore

---

### 2️⃣ Setup Backend
cd server  
npm install  
npm start

---

### 3️⃣ Setup Frontend
cd client  
npm install  
npm start

---

## 🌐 API Endpoints

| Method | Endpoint        | Description        |
|--------|---------------|--------------------|
| GET    | /books        | Get all books      |
| POST   | /books        | Add a new book     |
| PUT    | /books/:id    | Update a book      |
| DELETE | /books/:id    | Delete a book      |
